Linux Detection Engineering Lab

This project is a practical Linux security lab focused on attack simulation, log analysis, detection rules, and MITRE ATT&CK mapping.

Planned scenarios:

SSH brute-force detection
Suspicious cron job creation
SSH authorized_keys modification

Technologies:

Ubuntu Linux
Wazuh
Sigma
MITRE ATT&CK
Bash

Project status: Work in progress.

This project is intended only for educational purposes and authorised laboratory environments.
